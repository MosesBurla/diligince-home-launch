# Service Vendor Projects Module

## Overview
The Projects module allows Service Vendors to track and manage projects resulting from accepted quotations and purchase orders. Monitor milestones, manage deliverables, and communicate with Industry clients.

## Pages & Navigation

| Page | Route | Description |
|------|-------|-------------|
| All Projects | `/dashboard/service-vendor-projects` | Overview of all projects |
| Active Projects | `/dashboard/service-vendor-projects/active` | Currently active projects |
| Completed Projects | `/dashboard/service-vendor-projects/completed` | Finished projects |
| Project Details | `/dashboard/vendor/projects/:id` | Full project view |

## Key Features
- **Project Tracking** — Monitor project status and progress
- **Milestone Management** — Track and update delivery milestones
- **Deliverables** — Manage deliverable items and mark completion
- **Client Communication** — Communicate with Industry buyers about project progress
- **Document Upload** — Upload project deliverables and reports
- **AI-Powered Search** — Search across project listings

## How To: View Active Projects
### Steps
1. Navigate to [Active Projects](/dashboard/service-vendor-projects/active)
2. Browse your current projects
3. Click on a project to view [details](/dashboard/vendor/projects/:id)

## How To: Update Project Progress
### Steps
1. Open a project from [All Projects](/dashboard/service-vendor-projects)
2. Navigate to the milestone/task section
3. Update task statuses and completion percentages
4. Post progress updates for the client
5. Upload deliverable documents as needed

## Common Questions

**Q:** Where are my active projects?
**A:** Go to [Active Projects](/dashboard/service-vendor-projects/active).

**Q:** How do I update a project milestone?
**A:** Open the project from [All Projects](/dashboard/service-vendor-projects) and update milestones on the detail page.

**Q:** Where are completed projects?
**A:** View them at [Completed Projects](/dashboard/service-vendor-projects/completed).

**Q:** How do I communicate with the client about a project?
**A:** Use the project detail page updates section or go to [Messages](/dashboard/service-vendor-messages).
