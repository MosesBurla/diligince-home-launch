# Service Vendor Messages Module

## Overview
The Messages module enables Service Vendors to communicate with Industry clients, discuss RFQs, negotiate quotations, and coordinate on active projects.

## Pages & Navigation

| Page | Route | Description |
|------|-------|-------------|
| Messages | `/dashboard/service-vendor-messages` | Main messaging interface |

## Key Features
- **Direct Messaging** — Chat with Industry clients
- **Contextual Conversations** — Messages linked to specific RFQs, quotations, or projects
- **File Sharing** — Share documents and files in conversations
- **Notifications** — Real-time alerts for new messages
- **Message History** — Full conversation history and search

## How To: Message a Client
### Steps
1. Navigate to [Messages](/dashboard/service-vendor-messages)
2. Select an existing conversation or start a new one
3. Type your message and click Send
4. Attach files using the attachment icon if needed

## Common Questions

**Q:** Where do I message Industry clients?
**A:** Go to [Messages](/dashboard/service-vendor-messages).

**Q:** Can I send files to clients?
**A:** Yes, use the attachment icon in the [Messages](/dashboard/service-vendor-messages) chat.

**Q:** Are messages linked to specific projects?
**A:** Yes, contextual messages are tagged with the related RFQ, quotation, or project.
