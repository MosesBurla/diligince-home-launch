# Industry Stakeholders Module

## Overview
The Stakeholders module helps Industry users manage their relationships with vendors and professionals they've worked with or are currently working with.

## Pages & Navigation

| Page | Route | Description |
|------|-------|-------------|
| Vendors | `/dashboard/stakeholders/vendors` | List of all vendor relationships |
| Professionals | `/dashboard/stakeholders/professionals` | List of all professional relationships |

## Key Features
- **Vendor Directory** — View all vendors you've interacted with
- **Professional Directory** — View all professionals you've engaged
- **Relationship Tracking** — Track engagement history with each stakeholder
- **AI-Powered Search** — Search across stakeholder listings
- **Status Filters** — Filter by active, inactive, or pending status

## How To: View Your Vendors
### Steps
1. Navigate to [Vendors](/dashboard/stakeholders/vendors)
2. Browse the list of vendors you've worked with
3. Use the AI Search Bar to find specific vendors
4. Click on a vendor to view their profile and engagement history

## Page Details

### Vendors
**Route:** `/dashboard/stakeholders/vendors`
**Description:** Directory of all vendor companies you have a relationship with.
**Available Actions:**
- Search using AI Search Bar
- Filter by specialization, status
- View vendor profiles
- Send messages

### Professionals
**Route:** `/dashboard/stakeholders/professionals`
**Description:** Directory of all individual professionals you've engaged.
**Available Actions:**
- Search using AI Search Bar
- Filter by expertise, status
- View professional profiles
- Send messages

## Common Questions

**Q:** Where can I see all my vendors?
**A:** Go to [Stakeholders - Vendors](/dashboard/stakeholders/vendors).

**Q:** Where can I see professionals I've worked with?
**A:** Go to [Stakeholders - Professionals](/dashboard/stakeholders/professionals).

**Q:** How is this different from Diligince HUB?
**A:** The [Diligince HUB](/dashboard/Diligince-hub/vendors) is for discovering new vendors; Stakeholders shows vendors/professionals you already have a relationship with.
