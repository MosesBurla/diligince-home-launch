# Industry Dashboard

## Overview
The Industry Dashboard is your central command center for all procurement activities. It provides real-time KPIs, analytics, pending actions, and quick access to common tasks.

## Pages & Navigation

| Page | Route | Description |
|------|-------|-------------|
| Industry Dashboard | `/dashboard/industry` | Main dashboard with KPIs and overview |

## Key Features
- **KPI Statistics Cards** — Total requirements, active POs, total spend, budget utilization
- **Procurement Analytics** — Charts showing requirements by status, trends over time, vendor performance
- **Budget Overview** — Allocated vs. spent vs. remaining budget breakdown
- **Pending Approvals Widget** — Items requiring your immediate action
- **Active Requirements Summary** — Quick view of in-progress requirements
- **Active Purchase Orders** — POs currently being fulfilled
- **Top Vendors** — Performance rankings of your vendors

## Page Details

### Industry Dashboard
**Route:** `/dashboard/industry`
**Description:** Central hub displaying all procurement metrics, charts, and action items at a glance.

**Available Actions:**
- View detailed statistics by clicking any KPI card
- Filter data using the date range picker
- Access pending approvals directly from the widget
- Quick-create a new requirement
- Navigate to any module via sidebar

## How To: View Dashboard Metrics
### Steps
1. Navigate to [Dashboard](/dashboard/industry)
2. View the KPI cards at the top for high-level stats
3. Scroll down to see procurement analytics charts
4. Review the budget overview section for spending insights
5. Check the pending approvals widget for items needing action

## How To: Take Action on Pending Items
### Steps
1. Go to [Dashboard](/dashboard/industry)
2. Locate the **Pending Approvals** widget
3. Click on any pending item to view details
4. Approve or reject the item as needed
5. Alternatively, go to [Pending Approvals](/dashboard/pending-approvals) for the full list

## Common Questions

**Q:** How do I see my procurement spending?
**A:** The [Dashboard](/dashboard/industry) shows a Budget Overview section with allocated, spent, committed, and remaining funds.

**Q:** Where can I see items waiting for my approval?
**A:** Check the Pending Approvals widget on the [Dashboard](/dashboard/industry), or go directly to [Pending Approvals](/dashboard/pending-approvals).

**Q:** Can I export dashboard reports?
**A:** Yes, use the export options on the [Analytics](/dashboard/industry-analytics) page for detailed reports.

**Q:** How do I create a new requirement quickly?
**A:** Click the "Create Requirement" quick action on the [Dashboard](/dashboard/industry), or go to [Create Requirement](/dashboard/create-requirement).
