# Industry Messages Module

## Overview
The Messages module is a communication hub for Industry users to interact with vendors, team members, and stakeholders. It supports direct messaging, group chats, contextual conversations linked to requirements or POs, and file sharing.

## Pages & Navigation

| Page | Route | Description |
|------|-------|-------------|
| Messages | `/dashboard/industry-messages` | Main messaging interface |

## Key Features
- **Direct Messaging** — One-on-one conversations with vendors
- **Group Chats** — Multi-party conversations for project teams
- **Contextual Messaging** — Conversations linked to specific requirements or POs
- **File Sharing** — Share documents, images, and files within chats
- **Notifications** — Real-time notifications for new messages
- **Search** — Search message history
- **Mentions** — @mention team members in conversations
- **Reactions** — React to messages with emojis

## How To: Send a Message to a Vendor
### Steps
1. Navigate to [Messages](/dashboard/industry-messages)
2. Select an existing conversation or start a new one
3. Choose the vendor from your contacts
4. Type your message and click Send
5. Attach files if needed using the attachment icon

## How To: Start a Contextual Conversation
### Steps
1. Open a requirement or PO detail page
2. Click "Message Vendor" or the chat icon
3. The conversation is automatically linked to that requirement/PO
4. Messages will appear in [Messages](/dashboard/industry-messages) with the context tag

## Common Questions

**Q:** Where do I message vendors?
**A:** Go to [Messages](/dashboard/industry-messages) to access all your conversations.

**Q:** Can I attach files to messages?
**A:** Yes, use the attachment icon in the chat to share documents and files.

**Q:** How do I know if I have new messages?
**A:** You'll receive real-time notifications, and unread messages are indicated in the sidebar.

**Q:** Can I link a conversation to a requirement or PO?
**A:** Yes, start a conversation from the requirement or PO detail page for automatic linking, or use contextual messaging.
