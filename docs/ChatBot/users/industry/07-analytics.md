# Industry Analytics Module

## Overview
The Analytics module provides Industry users with reports, procurement insights, and data visualizations to help make informed procurement decisions.

## Pages & Navigation

| Page | Route | Description |
|------|-------|-------------|
| Analytics | `/dashboard/industry-analytics` | Reports, charts, and procurement insights |

## Key Features
- **Procurement Reports** — Detailed reports on spending, requirements, and vendor performance
- **Data Visualizations** — Charts and graphs for trends analysis
- **Budget Tracking** — Visual budget utilization and forecasting
- **Vendor Performance** — Ratings and performance metrics across vendors
- **Export** — Export reports in PDF, XLSX, CSV formats
- **Date Filtering** — Filter analytics by custom date ranges

## How To: View Procurement Reports
### Steps
1. Navigate to [Analytics](/dashboard/industry-analytics)
2. Select the report type from the available options
3. Set the date range using the date picker
4. View charts and data visualizations
5. Export reports using the export button

## Common Questions

**Q:** Where can I see procurement reports?
**A:** Go to [Analytics](/dashboard/industry-analytics) for all reports and insights.

**Q:** Can I export analytics data?
**A:** Yes, export in PDF, XLSX, or CSV format from the [Analytics](/dashboard/industry-analytics) page.

**Q:** How do I track vendor performance?
**A:** The [Analytics](/dashboard/industry-analytics) page includes vendor performance rankings and metrics.

**Q:** Can I filter reports by date?
**A:** Yes, use the date range picker on the [Analytics](/dashboard/industry-analytics) page to filter data.
