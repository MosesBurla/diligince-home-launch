# Industry Documents Module

## Overview
The Documents module provides centralized document management for Industry users. Upload, organize, categorize, and manage all procurement-related documents in one place.

## Pages & Navigation

| Page | Route | Description |
|------|-------|-------------|
| Documents | `/dashboard/industry-documents` | Document management center |

## Key Features
- **Document Upload** — Upload documents of various file types
- **Categorization** — Organize documents by category (contracts, specs, compliance, etc.)
- **AI-Powered Search** — Search across all documents
- **Version Control** — Track document versions
- **Status Tracking** — Active, archived, expired document statuses
- **Linked Documents** — Documents linked to requirements, POs, or projects

## How To: Upload a Document
### Steps
1. Navigate to [Documents](/dashboard/industry-documents)
2. Click "Upload" or "Add Document"
3. Select the file from your computer
4. Choose a category and add tags
5. Set access permissions if needed
6. Click "Save" to upload

## How To: Search Documents
### Steps
1. Go to [Documents](/dashboard/industry-documents)
2. Use the AI Search Bar to search by name, category, or tag
3. Click the search icon or press Enter
4. Browse results in the document list

## Common Questions

**Q:** Where do I manage my documents?
**A:** Go to [Documents](/dashboard/industry-documents) for all document management.

**Q:** What file types can I upload?
**A:** Common document types include PDF, Word, Excel, images, and CAD files.

**Q:** Can I link documents to a requirement or PO?
**A:** Yes, documents can be linked to specific requirements, purchase orders, or projects during upload or from their detail pages.
